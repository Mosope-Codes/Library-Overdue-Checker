Library Overdue Checker


An app that computes the total charges for overdue library books. Let's do this, shall we?
For each library book, you can enter the due date and (optionally), the overdue charge per day, the maximum overdue charge, and of course, the title.
If the optional values are not entered, then the preset values are used.
A complete list of the book information is displayed when you're done entering the input data.
You can also enter different return dates to compare the overdue charges.
